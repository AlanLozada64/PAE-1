2. Carga del Dataset desde API Pblica (Open-Meteo Archive)

# Definir la URL base y los parmetros de la consulta
API_URL = "https://archive-api.open-meteo.com/v1/archive"
LAT = 52.52      # Berln
LON = 13.41
START_DATE = "2023-01-01"
END_DATE = "2025-11-18" # Fecha actual para incluir datos de 2025

params = {
    "latitude": LAT,
    "longitude": LON,
    "start_date": START_DATE,
    "end_date": END_DATE,
    # Variables a extraer
    "daily": ["temperature_2m_max", "precipitation_sum", "wind_speed_10m_max"],
    "timezone": "Europe/Berlin"
}

print(f"Consultando datos histricos de Berln ({START_DATE} a {END_DATE})...")
# Realizar la solicitud HTTP
try:
    response = requests.get(API_URL, params=params)
    response.raise_for_status() 
    data = response.json()
    
    # Transformar la respuesta JSON a un DataFrame
    daily_data = data.get('daily')
    df = pd.DataFrame({
        'Fecha': pd.to_datetime(daily_data['time']),
        'Temp_Maxima_C': daily_data['temperature_2m_max'],
        'Precipitacion_mm': daily_data['precipitation_sum'],
        'Viento_Max_kmh': daily_data['wind_speed_10m_max']
    }).set_index('Fecha')

    # ESTADO INICIAL DEL DATASET
    print(f"\n[INFO]: Dataset cargado con {len(df)} registros diarios.")
    print("Primeras filas del DataFrame (df.head()):")
    print(df.head())

except requests.exceptions.RequestException as e:
    print(f"Error al conectar con la API: {e}")
    print("Generando un DataFrame de prueba para continuar con el EDA (FALLBACK)...")
    # Creando un DataFrame de emergencia
    periods = (pd.to_datetime(END_DATE) - pd.to_datetime(START_DATE)).days + 1
    data_test = {
        'Fecha': pd.to_datetime(pd.date_range(start=START_DATE, periods=periods, freq='D')),
        'Temp_Maxima_C': np.random.uniform(-5, 35, periods),
        'Precipitacion_mm': np.random.uniform(0, 20, periods),
        'Viento_Max_kmh': np.random.uniform(5, 55, periods)
    }
    df = pd.DataFrame(data_test).set_index('Fecha')
    print(f"\n[INFO]: Dataset cargado con {len(df)} registros diarios (DataFrame de prueba).")
    print(df.head())

4. Transformacin de Datos (Creacin de Categrica)

Esta celda realiza las transformaciones de datos necesarias para el anlisis: 1) Extrae el 'Ao' del ndice de fecha para agrupar y comparar interanualmente. 2) Crea la variable categrica Categoria_Temp ('Frio', 'Templado', 'Calido') a partir de la columna numrica Temp_Maxima_C, lo cual es un requisito de la gua de laboratorio. Finalmente, agrupa por ao para obtener estadsticas interanuales.

In[8]:
if not df.empty:
    # **A) CREAR VARIABLES TEMPORALES Y CATEGRICAS PARA EDA:**
    print("--- Transformacin: Creacin de variable 'Ao' y 'Categoria_Temp' ---")
    
    # 1. Variable 'Ao' para la comparacin interanual
    df['Ao'] = df.index.year
    
    # 2. Variable categrica 'Categoria_Temp'
    def categorizar_temperatura(temp):
        if temp < 5:
            return 'Frio'      
        elif 5 <= temp < 20:
            return 'Templado' 
        else:
            return 'Calido'    

    df['Categoria_Temp'] = df['Temp_Maxima_C'].apply(categorizar_temperatura)
    print("Conteo de das por categora de temperatura (total 2023-2025):")
    print(df['Categoria_Temp'].value_counts())
    
    # Anlisis interanual de tendencias para el resumen
    analisis_interanual = df.groupby('Ao').agg(
        Total_Dias=('Temp_Maxima_C', 'count'),
        Temp_Maxima_Promedio=('Temp_Maxima_C', 'mean'),
        Precipitacion_Total=('Precipitacion_mm', 'sum'),
        Viento_Max_Promedio=('Viento_Max_kmh', 'mean')
    ).sort_values(by='Ao', ascending=True)

    analisis_frecuencia = df.groupby('Ao')['Categoria_Temp'].value_counts().unstack(fill_value=0)
    print("\n[Tabla 1] Estadsticas Promedio Agrupadas por Ao:")
    print(analisis_interanual)
else:
    print("Saltando Transformacin de Datos: DataFrame vaco.")

5. Visualizacin de Datos (Matplotlib) - EDA Requerido

Esta celda genera las seis visualizaciones clave del EDA, incluyendo los cuatro tipos mnimos requeridos por la gua de laboratorio (Histograma, Boxplot, Grfico de Barras Categrico, Grfico de Dispersin) y dos grficos de resumen adicionales. La funcin principal es visualizar la distribucin, detectar outliers, observar tendencias interanuales y explorar patrones entre variables.

if not df.empty:

    # Paleta de colores para las visualizaciones
    colores = ['#4c72b0', '#55a868', '#c44e52', '#8172b2', '#ccb974', '#64b5cd']

    # 1. --- A) Histograma (Distribucin) ---
    axes[0, 0].hist(df['Temp_Maxima_C'], bins=30, color=colores[0], edgecolor='white', alpha=0.8)
    axes[0, 0].axvline(df['Temp_Maxima_C'].mean(), color='red', linestyle='dashed', linewidth=2, label='Media')
    axes[0, 0].set_title('A) Histograma: Distribucin de Temperatura Mxima (C)', fontsize=15)
    axes[0, 0].set_xlabel('Temperatura Mxima (C)')
    axes[0, 0].set_ylabel('Frecuencia de Das')
    axes[0, 0].legend()
    
    # 2. --- B.1) Boxplot (Deteccin de Outliers y Dispersin) ---
    box = df[['Temp_Maxima_C', 'Ao']].boxplot(by='Ao', ax=axes[0, 1], patch_artist=True, return_type='dict', medianprops={'color': 'black'})
    for patch, color in zip(box['Temp_Maxima_C']['boxes'], [colores[1], colores[2], colores[3]]):
        patch.set_facecolor(color)
    axes[0, 1].set_title('B) Boxplot: Temperatura Mxima por Ao', fontsize=15)
    axes[0, 1].set_xlabel('Ao')
    axes[0, 1].set_ylabel('Temperatura Mxima (C)')
    plt.suptitle('') # Eliminar subttulo duplicado del boxplot

    # 3. --- C) Grfico de Barras (Categora) ---
    order_cols = ['Frio', 'Templado', 'Calido']
    frecuencia_total = df['Categoria_Temp'].value_counts().reindex(order_cols, fill_value=0)
    
    frecuencia_total.plot(ax=axes[1, 0], kind='bar', rot=0, color=[colores[0], colores[1], colores[2]], alpha=0.9)
    axes[1, 0].set_title('C) Grfico de Barras: Frecuencia Total por Categora de Temperatura', fontsize=15)

    axes[1, 0].grid(axis='y', alpha=0.5)

    # 4. --- D) Grfico de Dispersin (Observacin de Patrn) ---

    axes[1, 1].set_title('D) Grfico de Dispersin: Temperatura Mxima vs. Precipitacin (mm)', fontsize=15)

6. Conclusiones del EDA y Resumen Funcional

Conclusiones del Anlisis Exploratorio de Datos (EDA)

1. Estructura y Limpieza (Punto 3)

  • Calidad de los Datos: El dataset real extrado de la API es de alta calidad, con cero valores faltantes. No se requiere limpieza ni imputacin.
  • Tipos de Variables: Se identifican 3 variables numricas (Temp_Maxima_C, Precipitacion_mm, Viento_Max_kmh) y una variable categrica creada (Categoria_Temp).

2. Tendencias Centrales y Dispersin (Punto 3)

  • Temperatura: Media y mediana similares indican distribucin centrada, pero la alta desviacin estndar y outliers confirman alta variabilidad estacional.
  • Precipitacin: Media (2.1 mm) muy superior a la mediana (0.6 mm). Esto indica que la mayora de das son secos, y la media se eleva por eventos extremos.

3. Interpretacin de Visualizaciones (Punto 5)

  • A) Histograma: La temperatura muestra una distribucin bimodal, reflejando la estacionalidad.
  • B) Boxplot: Se observa una tendencia clara de calentamiento entre 2023 y 2025. Los outliers superiores representan olas de calor.
  • C) Grfico de Barras: Predomina la categora "Templado", pero hay menos das fros y ms das clidos con el tiempo.
  • D) Dispersin: La precipitacin no correlaciona con la temperatura. Los das secos dominan todo el rango trmico.

Conclusin General del EDA

Existe una tendencia robusta al calentamiento en Berln (20232025). Temperatura y precipitacin muestran alta volatilidad y eventos extremos. El viento es la variable ms estable. Esta informacin es til para planificacin energtica, hdrica y mitigacin de riesgos climticos.
